Featured Projects Things that I have worked/working on
  • PostOptic

    It is a WIP API for Optical Character Recognition (OCR) technologies like tesseract-ocr that simplifies the image's post processing to enhance the character recognition of the image. It also features multi-level post-processing which allows you to post-process an already processed image.

    It is made in C# .NET Standard and has cross-platform support.

  • TricyCall

    This is our capstone project that follows the modified MERN stack (MondoDB, Express.JS, React Native, Node JS.) It is a hailing application similar to Grab and Uber but suited for tricycles. It is also cloud-based, which ensures that the uptime reliability is at a satisfactory level.

    During development, I was responsible for the whole backend of the system and the frontend with the exception of mobile UI design.

  • OsuPlaylistCreator

    It is a simple tool that retrieves osu!'s "Songs" folder and creates a playlist for it.

  • QMunity

    During pandemic, every subdivisions has to identify each person who come in or out for tracking purposes. QMunity is a web application that allows easy logging for people who comes in/out of the subdivision. The logged data is also served in a spreadsheet which allows easier data encoding.

    It was written in Node JS, with the use of Socket.io as a library to use for real-time data and communication. I was the person in charge of the backend.

  • DogReg (Dog Registration System)

    It is a client-based project for our school research that resolves one of the recurring problems of our client (barangay): dog management. This web systems encourages people inside the client's area to register their pet dogs which can be helpful on scenarios such as if the dog suddenly went missing.

    I am responsible for the whole backend of this system, which is done in ASP.NET MVC.

  • Floob

    Floob is a bot trainer for the flash game called "Adventure Quest Worlds". This application allows you to create a script that automates essentially anything you want such as item farming, quest storyline completion, etc.

    It is written in C# .Net Framework 4.8 which makes it available only for Windows but supports up to windows 7.
